### // about

I work across both sides of security — breaking into systems to find what an attacker would find, and investigating incidents after the fact to reconstruct exactly what happened. My background spans **VAPT**, **digital forensics & cyber crime investigation**, **SOC monitoring (Splunk)**, and independent **malware/threat research** — backed by a defence-and-counter-terrorism analyst background applied to civilian security work.

- 🔴🔵 **Purple Team** — offense and defense as one discipline, not two separate tracks
- 🔍 Active lab: [`purple-team-lab`](https://github.com/Founder-neeravpatel/purple-team-lab) — red team, blue team, forensics, and OSINT methodology documented and practiced
- 🛠️ Building: [`vapt-recon-tool`](https://github.com/Founder-neeravpatel/vapt-recon-tool) — automated recon + live CVE correlation
